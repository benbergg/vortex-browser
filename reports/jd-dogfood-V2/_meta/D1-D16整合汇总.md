## 4. 风险点 + 诚实标注 (V2 重测版)

- **京东 3C 搜索 FCP=116960ms (116 秒!) 是 V2 重测真发现**: 不是评测误差, 京东搜索页 SPA 客户端渲染 + 京东埋点干扰, **真**是京东平台级性能问题
- **京东 LCP 元素未触发**: LCP entries=[] (京东 SPA 详情页 LCP 数据不完整, 与 V2 早期 3C 详情 LCP=3216ms 不一致, vortex 注入差异)
- **京东 D12/D15 探针沿用淘宝结论**: 京东 `navigator.onLine` / `matchMedia` 行为与淘宝同 (W3C 标准), 假断网 / 暗黑切换方法不可行, 跨平台同结论
- **R8 白盒核对发现** (跨双平台 V2 真发现):
  - **vortex_observe 在 main frame 0 全部扫空** (ReferenceError: applyReactClickableMarker is not defined) — 跨双平台 + example.com 一致, **不是** 平台问题, **是** vortex 自身 bug
  - **vortex_debug_read.filter 子字段未文档化 + handler 字段名不统一** — V2 D16 ROI 最高发现
- **路径选择** (按 `ecom-anti-detection` skill): 跨域直接 navigate 商品 URL 触发风控, V2 重测版用京东首页 input + Enter 真实搜索 + mouse_drag 真实点击, 走 SPA 站内路由 (带 referer + cookies + from=pc_search_sd 模拟人)
- **跨域直接 navigate 触发风控案例**: V2 重测版第二次 `navigate search.jd.com/Search?keyword=空调` 被京东重定向到 `www.jd.com/?from=pc_search_sd` (搜索引擎防刷); 用首页 input + Enter 模拟人搜索绕过
